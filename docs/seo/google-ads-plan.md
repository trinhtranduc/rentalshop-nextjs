## Settings quan trọng

- **Vị trí:** Chỉ Việt Nam (ưu tiên HCM, Hà Nội, Đà Nẵng)
- **Giờ chạy:** 7h - 23h (tắt 23h-7h)
- **Thiết bị:** Cả mobile + desktop (monitor CTR theo device)
- **Bidding:** Manual CPC (vì budget nhỏ, cần kiểm soát)
- **Rotation:** Optimize for clicks (tuần đầu), sau đổi Optimize for conversions
- **Conversion tracking:** Setup event đăng ký thành công trên anyrent.shop
